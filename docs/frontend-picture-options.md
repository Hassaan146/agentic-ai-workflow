# Frontend Picture Options

These are visual directions we can use when you give the detailed frontend prompt. Each option can become a generated hero image, 3D scene, or animated website direction.

## Option 1: Mission Control Graph

Hero visual:

- A dark interface with a glowing state graph in the center.
- Nodes pulse as tasks move from one agent to another.
- Side panels show logs, confidence, and output status.

Best for:

- Showing Agentic AI Workflow as a serious orchestration platform.

Image prompt:

```text
Futuristic AI mission control interface, central glowing state graph with connected nodes, active task routing lines, dark neutral UI, one teal execution color, cinematic lighting, high-detail dashboard, no people, 16:9, 4K
```

## Option 2: Agent Command Table

Hero visual:

- A top-down command table where specialized AI agents are represented as labeled cards.
- The user request enters from the left and becomes structured tasks.
- The final answer leaves on the right.

Best for:

- Explaining the workflow clearly to recruiters and users.

Image prompt:

```text
Top-down product visualization of AI agent cards on a command table, user request transforming into structured task cards, arrows showing prerequisite order, clean SaaS interface, subtle depth, high contrast, 4K
```

## Option 3: Living Workflow Canvas

Hero visual:

- A large canvas with draggable nodes, conditional edges, and execution traces.
- Looks like a builder tool rather than a chatbot.

Best for:

- Positioning the product as a future visual workflow studio.

Image prompt:

```text
Premium workflow builder canvas for AI agents, draggable graph nodes, conditional routing edges, live execution timeline, polished SaaS dashboard, restrained colors, crisp typography, 4K
```

## Recommended Direction

Start with **Option 1: Mission Control Graph** for the first public version.

Reason:

- It instantly communicates multi-agent orchestration.
- It feels more advanced than a chatbot.
- It matches the current backend concept: state graph, agents, traces, verification.
